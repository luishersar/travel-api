# Travel Request API

Una API construida en ASP.NET Core con autenticación JWT y Entity Framework Core que permite a los usuarios registrar y gestionar solicitudes de viaje.

---

## 🚀 Tecnologías utilizadas

- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger (Swashbuckle)

---

## ⚙️ Configuración del proyecto

Se debe clonar el proyecto y ejecutar localmente para poder correr migraciones y tener las tablas que almientarán los endpoints desarrollados.
